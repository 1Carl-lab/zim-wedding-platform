# Zimbabwe Wedding Platform - Frontend

This is the React.js frontend application for the Zimbabwe Wedding Platform.

## Features
- Vendor Listings: Browse vendors by category (planners, caterers, photographers, venues, etc.)
- Search and Filter: Search vendors by location, ratings, and availability
- Verified Reviews: View and submit reviews for vendors
- Shortlisting: Create and manage shortlists of preferred vendors
- Booking System: Submit booking inquiries to vendors

## Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### Building for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Technology Stack
- React.js 18
- React Router for navigation
- CSS/styled-components for styling
